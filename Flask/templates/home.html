Hawaii weather station Data for Developers

API Requests can be obtained for the following

Data set is from 2010-01-01 till 2017-08-23. All output will be in JSON Format.
  • To get the precipitation of the stations with the dates as key.

    • Use /api/v1.0/precipitation after the host. example: //api/v1.0/precipitation

  • To get the list of the stations

    • Use /api/v1.0/stations after the host. example: //api/v1.0/stations

  • To get the temperature of the stations with dates as key

    • Use /api/v1.0/tobs after the host. example: //api/v1.0/tobs

  • To get minimum Average and Maximum Temp of the stations

    • Use /api/v1.0/{start_date} after the host.Here parameter to be entered is start date(YYYY-MM-DD) format. Its will provide all data from start date. example: //api/v1.0/2017-01-03

  • To get minimum Average and Maximum Temp of the stations

    • Use /api/v1.0/{start_date}/{end_date} after the host.Here parameters to be entered is "start_date / end_date"(YYYY-MM-DD) format. Its will provide all data from start till end date. example: //api/v1.0/2016-01-03/2017-01-01
